Welcome to jjm_utils's documentation!
=====================================

**jjm_utils** is a collection of scripts and tools to automate and simplify
the use of our HPC cluster to process and analyze neuroimaging data.

.. note::

    This project is actively developed, but can be low priority. If you
    need information not provided here, prompt Mike to put it up.
